windows下hugo快速构建博客

Aug 30, 2020
1 min read

windows

安装运行

  • https://github.com/gohugoio/hugo/releases 下载 windows-64bit.zip
  • cd e:
  • mkdri hugo
  • cd hugo
  • mkdir bin
  • 解压缩到e:\hugo\bin
  • hugo version
  • hugo new site myblog
  • cd myblog
  • https://themes.gohugo.io
  • git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c
  • cd themes
  • cd ..
  • hugo server -t m10c –buildDrafts
  • http://localhost:1313/
  • hugo new post/blog.md
  • cd content
  • cd post
  • 用vscode 来写 blog.md
  • cd myblog
  • hugo server -t m10c –buildDrafts
  • 修改 title

部署到github

  • create a new repository
    • Owner
      • frzw
    • Repository name
      • frzw.github.io(必须是Owner名字+github.io)
  • cd myblog
  • hugo –theme=m10c –baseUrl="https://frzw.github.io/” –buildDrafts
  • cd public
  • git init
  • git add.
  • git commit -m “我的hugo博客第一次提交”
  • git remote add origin https://github.com/frzw/frzw.github.io.git
  • git push -u origin master
